pingora 是什么

pingora 是 cloudflare 公司开源的一个代理框架，本意是为了弥补一些 nginx 的不足。 我研究 pingora 的原因纯粹是想学习一下 rust，要说 pingora 能不能很好的替代 nginx， 还得看大家的需求。

从 pingora 的快速文档可以看出来 pingora 其实只是开源了一个可变成的 http 服务框架， 有兴趣的朋友可以按照教程跑一下。我自己因为跑不通他们的 1.1.1.1 的服务器，所以没成功。 但是他是自带了一个代理服务器需要的组件。所有的代码我还没有研究过，本文仅解释一下被用来 实现反向代理的组件
